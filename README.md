# EXAPUNKS_RUSIFIER

Русификатор EXAPUNKS от Vahe.

## Как использовать

1. При необходимости открой `config.json` и проверь `game_dir`
2. Один раз запусти `run.cmd`
3. После установки запускай игру обычным `EXAPUNKS.exe` из папки игры

## Что делает `run.cmd`

- создает резервную копию в `_ru_backup`
- копирует переведенные файлы
- устанавливает безопасный `EXAPUNKS.exe`, который запускает оригинальную игру без отдельного лаунчера

## Откат

1. Запусти `restore.cmd`
2. Будет восстановлена последняя резервная копия из `_ru_backup`

## Что лежит в корне

- `run.cmd` - установить русификатор
- `restore.cmd` - откатить последнюю резервную копию
- `config.json` - путь к папке игры
- `README.txt` - краткая инструкция

## Что скрыто

- внутренняя реализация лежит в скрытой папке `_internal`
